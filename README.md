## Run Locally

Clone the project

```bash
  git clone https://github.com/Scraayp/kleren.michalk.nl.git
```

Go to the project directory

```bash
  cd kleren.michalk.nl
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React (NextJS), TailwindCss

**Server:** Node

**Hosting:** Vercel
